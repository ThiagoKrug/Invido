Invido
======

Tente adivinhar o valor do invido!

Requisitos necessários:
- Possuir um servidor web (apache, ngix, etc)
- Possuir o java runtime edition

Como instalar:
- Instale o IzPack <http://izpack.org/>
- Baixe o repositório do projeto no link https://github.com/ThiagoKrug/Invido
- Acesse pelo terminal/cmd a pasta onde você baixou o projeto
- Execute o comando "compile install.xml -b . -o install.jar -k standard"
- Execute o arquivo install.jar que foi criado
- Siga as instruções, e instale o software na pasta de projetos de seu servidor web (apache, ngix, etc)

Instruções de como jogar:
- Acesse pelo navegador em caminho/para/oprograma/Invido/public/
- Informe o valor que você quer adivinhar que será o invido
- Clique em Chutar!
- Se acertar, seu placar de acertos irá aumentar
- Se errar, seu placar de erros irá aumentar
- O jogo termina quando não existirem mais cartas para ser compradas
- Você pode iniciar um novo jogo quando terminar o jogo atual
