Invido
======

Tente adivinhar o valor do invido!

Requisitos necessários:
- Possuir um servidor web (apache, ngix, etc)
- Possuir o java runtime edition

Como instalar:
1- Instale o IzPack <http://izpack.org/>
2- Baixe o repositório do projeto no link https://github.com/ThiagoKrug/Invido
3- Acesse pelo terminal/cmd a pasta onde você baixou o projeto
4- Execute o comando "compile install.xml -b . -o install.jar -k standard"
5- Execute o arquivo install.jar que foi criado
6- Siga as instruções, e instale o software na pasta de projetos de seu servidor web (apache, ngix, etc)

Instruções de como jogar:
1- Acesse pelo navegador em caminho/para/oprograma/Invido/public/
2- Informe o valor que você quer adivinhar que será o invido
3- Clique em Chutar!
4- Se acertar, seu placar de acertos irá aumentar
5- Se errar, seu placar de erros irá aumentar
6- O jogo termina quando não existirem mais cartas para ser compradas
7- Você pode iniciar um novo jogo quando terminar o jogo atual
